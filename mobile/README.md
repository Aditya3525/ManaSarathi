# ManaSarathi Mobile App (Future Scope)

The mobile application is planned as a future extension of the current web + backend platform.

## Planned Direction

- React Native + Expo architecture aligned with the existing backend APIs.
- Shared domain flows with the web platform: auth, chat, assessments, mood, progress, and recommendations.
- Secure token storage and optional biometric login.
- Gradual rollout with feature parity milestones.

## Target Milestones

1. Foundation and authentication flows.
2. Dashboard, mood tracking, and assessment journeys.
3. Chat, recommendations, and content/practice experiences.
4. Advanced notifications, analytics, and offline reliability.

## Notes

- This directory remains reserved for the mobile track.
- The current production focus is web frontend + backend API.
